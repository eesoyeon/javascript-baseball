# ⚾️ 숫자 야구 게임

컴퓨터가 무작위로 생성한 3자리 숫자를 플레이어가 추측하여 맞추는 게임입니다.

각 추측마다 컴퓨터는 플레이어에게 "스트라이크"와 "볼"의 개수, 또는 "낫싱" 정보를 알려주며, 플레이어는 이를 바탕으로 정답을 유추해 나가야 합니다.


## 📝 게임 규칙

1. 먼저, 플레이어는 컴퓨터에게 도전할 횟수를 입력합니다. (단, 1~15 사이의 숫자)

2. 컴퓨터는 `1에서 9 사이의 서로 다른 3자리 숫자`를 무작위로 생성합니다.

3. 플레이어는 3자리 숫자를 입력하고, 컴퓨터는 결과에 따라 "스트라이크"와 "볼"의 개수, 또는 "낫싱" 정보를 제공합니다.

    - `스트라이크`: 숫자와 위치가 모두 일치하는 경우
    - `볼`: 숫자는 맞지만 위치가 다른 경우
    - `낫싱`: 숫자가 모두 존재하지 않는 경우

4. 플레이어가 도전 횟수 안에 3자리 숫자를 맞추면, 플레이어가 승리합니다.

5. 플레이어의 시도 횟수가 도전 횟수를 넘어가게 되면, 컴퓨터가 승리합니다.

   
## 📥 게임 실행 방법

### "1"을 누르면, 플레이어가 도전 횟수를 설정하고 게임을 시작합니다.

게임이 시작되면 컴퓨터가 1부터 9까지 서로 다른 3자리 숫자를 선택하고, 플레이어는 그 숫자를 추측합니다.
